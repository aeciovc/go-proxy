Simple Proxy using golang
========================================================

Usage
====

Install dependencies:

.. code-block:: console

    go get gopkg.in/elazarl/goproxy.v1

Build:

.. code-block:: console
    
    go build -o bin/go-proxy main.go

Run:

.. code-block:: console
    
    ./bin/go-proxy $(PORT_NUMBER)
